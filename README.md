![Drag Racing](./App/images/image.png)

## How to install

Make a directory

```
mkdir [directory]
```

Clone this repository

```
git clone https://github.com/brunoaugustosilva/todosapp.git
```

Starts NPM

```
npm install
```

Tests

Run this project with:

```
npm start
```
